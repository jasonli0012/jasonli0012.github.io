Definition of a Group

Definition

A group is a set GGG, together with a map of G×GG\times GG×G into G (denoted g1∗g2g_1 * g_2g1​∗g2​) with some basic properties

Basic Properties

Closure 封闭性

∀g1,g2∈Gs.t.g1∗g2∈G\forall g_1, g_2\in G\quad s.t.\quad g_1 * g_2\in G ∀g1​,g2​∈Gs.t.g1​∗g2​∈G

从G×GG\times GG×G到GGG的映射称为群的积(Product)

Associativity 结合律

∀g1,g2∈Gs.t.g1∗(g2∗g3)=(g1∗g2)∗g3\forall g_1, g_2\in G\quad s.t.\quad g_1*(g_2*g_3) = (g_1*g_2)*g_3 ∀g1​,g2​∈Gs.t.g1​∗(g2​∗g3​)=(g1​∗g2​)∗g3​

Identity Element 单位元（幺元）

∀g∈G,∃e∈G,s.t.g∗e=e∗g=g\forall g\in G,\quad \exists e\in G,\quad s.t.\quad g*e=e*g=g ∀g∈G,∃e∈G,s.t.g∗e=e∗g=g

单位元eee是唯一的，单位元与群内元素的乘积为该元素本身。

Abelian and Inverse 阿贝尔群和逆元

∀g∈G,∃h∈G,s.t.g∗h=h∗g=e\forall g\in G, \quad \exists h\in G, \quad s.t. \quad g*h=h*g=e ∀g∈G,∃h∈G,s.t.g∗h=h∗g=e

阿贝尔群的群元素乘积的值与乘法运算时的次序无关，即阿贝尔群具备交换律；

群本身没有交换律存在，称上述公式中hhh为ggg的逆元(Inverse Element)，每个元素的逆元唯一存在。记群内元素ggg的逆元为g−1g^{-1}g−1

Groups’ basic properties

Given a set and an operation, there are four things that must be checked to show that this is a group:

  • Closure

  • Associativity

  • existence of an Identity

  • existence of Inverses

群可以被视为一个拥有封闭性，满足结合律且有单位元和逆元的二元运算的集合(G,∗)(G, *)(G,∗)，阿贝尔群满足交换律。

Some Examples of Groups

加法群

The integers under addition

Closure: ∀z1,z2∈Z,z1+z2∈z\forall z_1,z_2\in \mathbb{Z},\quad z_1+z_2\in\mathbb{z}∀z1​,z2​∈Z,z1​+z2​∈z

两个整数的和必然是一个整数，则(Z,+)(\mathbb{Z},+)(Z,+)满足封闭性

Associativity: addition is associative

加法必然满足结合律，则(Z,+)(\mathbb{Z},+)(Z,+)满足结合律

Identity: ∀z∈Z,0+z=z+0=z\forall z\in\mathbb{Z}, 0+z=z+0=z∀z∈Z,0+z=z+0=z

唯一存在单位元0，则(Z,+)(\mathbb{Z},+)(Z,+)存在单位元

Inverses: ∀z∈Z,∃(−z)∈Z,s.t.z+(−z)=(−z)+z=0\forall z\in\mathbb{Z},\exists(-z)\in\mathbb{Z},\quad s.t. z+(-z)=(-z)+z=0∀z∈Z,∃(−z)∈Z,s.t.z+(−z)=(−z)+z=0

每个元素存在唯一的逆元，则(Z,+)(\mathbb{Z},+)(Z,+)存在逆元
